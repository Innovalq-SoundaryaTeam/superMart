SuperMart – Image Replacement Guide

Complete list of every image in the project with the filename you should use.

📌 How to replace images

  1. Find the assets/images/ folder inside your project (it's already created)
  2. Rename your image file to exactly match the Save as filename column below
  3. Copy it into assets/images/
  4. Done — the website will automatically use it

✅ Images can be JPG or PNG — both formats work. If a JPG isn't found the site automatically tries PNG, and vice versa. Use the recommended size as a guide.
